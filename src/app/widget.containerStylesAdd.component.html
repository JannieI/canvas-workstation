Add a new Container Style
Help

Add a new Container Style, giving it a unique name. When creating or editing a Widget container or a text shape, select this name to apply the properties defined here to it. One can also link a Widget permanently to a style: it is applied (and re-applied) when the Widget is opened. The properties of the style will be used, but can still be changed on the Widget. This helps ensure that Widgets look the same across Dashboards, and also helps with bulk changes. Click the X or press Escape to close this form.
