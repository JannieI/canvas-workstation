Dashboard Help
This is your Dashboard on which data can be visualised. It can have more than one tab (like a spreadsheet), as shown at the bottom of the page. Some actions:
1. Data (called datasources) must be linked to a Dashboard before visualizing it. Data can come from files, databases or the web. In a larger company, this will be done by the responsible team. Once datasources have been added, you can add them to your dashboard.
If no datasources has been created before, add a new datasource: - Click Data in the top menu - Then select the type of source like From File). Once done, you will return to this Dashboard.
2. Link a datasource to this dashboard: - Click Data in the top menu - Then click Add Existing). Once done, return to this dashboard.
2. Add a Widget to visualise the added datasource. A widget can be a Graph, Table, Slicer (filter) or a Shape (text, circle, arrow, etc). Add a graph as follows: - Cclick Graph in the top menu - Then click Add Graph - Select the datasource, and drag fields onto the rows and columns - Return to this dashboard once done.
3. Format and arrange Widgets as necessary, using: - Arrange multiple widgets with the Arrange option in the top menu - Drag widgets by click, hold, drag and release - resize widgets with the drag handles on each corner and side.
4. All changes to a dashboard is saved automatically. These changes remain in draft format, and are not visible to other users. To make it visibile, Save your new Dashboard with a name using Dashboard -> Save menu options.
5. Existing Dashboards can be opened with Dashboard -> Open Existing menu options.
6. Collaboration (discussions with colleagues) happens via messages and tasks (click the Share icon in the top right menu).
7. Graphs and Table can be edited with the Graph -> Edit and Table -> Edit menu options respectively.
