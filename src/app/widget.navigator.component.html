Navigator
Help

Navigates / explores a web of relationships in a graphical fashion. One can start at any node (entity, like a company called Absa or a person called John), and then show the related nodes for a specific relationship (ie directorships of the company). After, one can proceed by clicking one of the children, or start with a totally different entity. There is no limit to the number of nodes, or the relationships per node. Click the X or press Escape to close this form.

Networks (web)
Help

A network is a pre-defined web of relationships. For example, it can be the relationships between companies, subsidaries, directors (executive and non-executive), shareholders, etc.
