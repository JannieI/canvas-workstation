Delete an existing Dashboard
You can only delete Dashboards to which you have access. Admin users have access to all Dashboards. It is not physically delete, and only moved to Trash (retaining its permissions). There is an option on the Open Dashboard form to retrieve a Dashboards previously moved to Trash - provided that one had access to it at the time it was trashed.
