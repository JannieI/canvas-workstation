Help

The existing open Dashboard will be closed (all changes will be saved automatically) and a new Dashboard will be opened. It has one Tab called Tab1 (shown in the Statusbar at the bottom of the screen). A Tab can be renamed by clicking on its name. More Tabs can be added using the in the Statusbar. You can add Dashboard properties (ie. Description) under the Dashboard -> Details menu option. After creation, the new Dashboard is only visible and editable by yourself. Other users or groups have to be granted Permissions, using the Dashboard -> Share menu option.
