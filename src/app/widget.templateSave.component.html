Help

This form registers the selected Widget as a Stored Widget Template. Once stored, it can be inserted on any Dashboard. They aim to simplify the process of creating new Widgets. Add a name and description (both are compulsory), and click Add. If you selected a Widget that has already been stored, the new Widget specification will be stored when you click Save. Widgets created previously from this template will NOT be affected. Click the X or press Escape to close this form.

The selected Widget will be added to the list of predefined Widget Templates. Widget Templates simplify the creation of Widgets on a Dashboard: the user simply selects a predefined Widget Template, which is then inserted into the Dashboard without any further work required. It automatically adds the relevant Datasource if not already added.
