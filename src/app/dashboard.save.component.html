Your security mode (set by Admin):

There are 2 security modes, the default is Normal
In Normal mode all users can do the following to all Dashboards that they have access to (full access is automatically granted to the creator, and further access can be granted to the same or lessor level that is owned): a. Open (for view in readonly mode - no changes can be done) b. Edit (a WIP version will be created, leaving Committed version unchanged) c. Discard (delete WIP version) d. Request-QA (on Request, the WIP -> Pending). This step is optional. e. Save (approves QA request and WIP/Pending -> Committed)
In Strict mode, there are different User roles:
1. Admin - can Open, Edit, Request-QA, QA and Save all Dashboards
2. DashboardBuilder, which is a type of group membership, can perform the following actions on all Dashboards that the group has access to: a. Open (readonly) b. Edit c. Discard d. Request-QA e. Save
3. All other Users can perform all actions to any Dashboard that the user has access to, BUT does not belong to a group.
