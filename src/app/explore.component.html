First add a datasource (click Data in the top menu). Then insert a Widget to visualise the data (click Create -> Widget in the top menu). Once the Widgets have been formatted and arranged, you can save the Dashboard with a name (File -> SaveAs). Existing Dashboards can be opened with File -> Open. Collaborating happens via messages and tasks (click user icon in the top right menu).
